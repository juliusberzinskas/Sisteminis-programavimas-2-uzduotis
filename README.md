Programa buvo sukurta vadovaujantis pirma užduotimi

Vartojimo instrukcija:
1. Paspauskite menu File -> Open
2. Pasirinktite txt. failą (studentaiXXXX.txt)
3. Studentų sarašas atsiras teksto lange
4. Spauskite generuoti
5. Studentai sugeneruoti skirtingose 'vargsiukai.txt' ir 'kietekai.txt' failuose
6. Norint uzdaryti programa spauskite Exit
